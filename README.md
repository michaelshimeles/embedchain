## NextJS 13 Starter Template 

## Tech Stack
NextJS 13 - Full Stack framework 

TailwindCSS - CSS framework

Shadcn UI - Component library

Clerk - Authentication

React Hook Form - Forms

Zod - Type Schema

Tanstack Query - Querying Data & State Management

Supabase - Database
